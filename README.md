# HR Analytics Dashboard (Power BI)

This project analyzes HR data to identify employee demographics, attrition patterns, and workforce distribution to support HR decision-making.

## About Data
This dataset contains employee-related information used for HR analytics, including demographics, employment status, and attrition data.

| Column Name        | Description                                             | Data Type |
| :----------------- | :------------------------------------------------------ | :-------- |
| employee_id        | Unique identifier for each employee                     | int       |
| age                | Employee age                                            | int       |
| gender             | Employee gender                                         | text      |
| department         | Department where the employee works                     | text      |
| job_role           | Employee job position                                   | text      |
| education_level    | Highest education level                                 | text      |
| hire_date          | Date the employee was hired                             | date      |
| termination_date   | Date the employee left the company (if applicable)      | date      |
| attrition          | Indicates whether the employee has left (Yes/No)        | text      |
| monthly_income     | Employee monthly salary                                 | double    |
| years_at_company   | Total years the employee has worked at the company      | int       |
| performance_rating | Employee performance evaluation score                   | int       |


## Tools
- Power BI
- MySQL
- Excel

## Key Insights
- Attrition trends by department and age group
- Employee distribution by gender and role
- Hiring and termination overview

## Process
- Data exploration and querying using SQL
- Data modeling and visualization in Power BI
- Dashboard creation for HR insights
